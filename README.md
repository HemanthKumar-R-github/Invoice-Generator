# 🧾 GST Invoice Generator

A modern, responsive GST Invoice Generator built using HTML, CSS, and JavaScript. The application allows users to create professional GST tax invoices, automatically calculate taxes, generate HSN-wise summaries, and download invoices as PDF files.

## ✨ Features

- 📄 Generate professional GST Tax Invoices
- 🏢 Seller and Buyer information management
- 📦 Dynamic product table
- ➕ Add or remove product rows
- 🧮 Automatic calculations for:
  - Taxable Amount
  - CGST
  - SGST
  - Grand Total
  - Round Off
- 📊 HSN-wise Tax Summary
- 💰 Amount in Words (Indian Numbering System)
- 🏦 Bank Details section
- ✍️ Declaration and Authorized Signatory
- 📥 Export invoice as PDF
- 📱 Fully responsive interface

---

## 📋 Invoice Features

### Seller Details

- Company Name
- Address
- GSTIN
- State
- Email
- Phone

### Buyer Details

- Buyer Name
- Address
- GSTIN
- State
- Place of Supply

### Invoice Details

- Invoice Number
- Invoice Date
- Delivery Note
- Buyer Order Number
- Dispatch Details
- Payment Terms

### Product Details

Each row supports:

- Product Description
- HSN/SAC Code
- Quantity
- Unit
- Rate
- GST Percentage

The application automatically calculates:

- Taxable Value
- CGST
- SGST
- Total Amount
